# Start-Up Procedure
1. Master Key **ON**.
2. Reverser **Neutral**.
3. Headlight Control Switch: **As Required**.
4. **Release** Doors.
5. **Close** Door Interlock.
6. Reverser **Forward**.
7. Power: **Min Brake**.
8. **Wait for brakes** to depressurize.
9. **Apply** Power.
